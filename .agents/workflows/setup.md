# Setup Project Workflow

This workflow sets up both the backend and frontend dependencies for the RestoCart-AI project.

### 1. Backend Setup
1. Change directory to backend: `cd backend`
2. Install dependencies and setup environment with uv: `uv sync`

### 2. Frontend Setup
1. Change directory to frontend: `cd frontend`
2. Install dependencies: `npm install`
