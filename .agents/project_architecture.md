# Project Architecture

This is a **simplified e-commerce demo** called **RestoCart AI**.

## 📁 Repository Structure

```text
restocart-ai/
├── backend/       # FastAPI Backend (Python)
├── frontend/      # React + Vite Frontend
├── .agents/       # Antigravity AI Automation
└── README.md      # Main documentation
```

## ⚙️ Running Locally

1. Set up `GROQ_API_KEY` in `backend/.env`.
2. Run `/all` to setup and launch the application.
