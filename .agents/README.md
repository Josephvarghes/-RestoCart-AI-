# .agents Folder

This folder contains configurations and workflows used by **Antigravity** (AI Assistant) to help you manage the RestoCart-AI project more efficiently.

## Available Workflows

- **/setup**: Sets up the project environment (backend and frontend dependencies).
- **/run**: Runs the backend (FastAPI) and frontend (Vite) servers.
- **/all**: Automatically setup and run everything.

## Project Structure (Summarized)

- **backend/**: FastAPI application for the e-commerce demo.
- **frontend/**: React + Vite application for the user interface.

## Getting Started

1. Set up the `GROQ_API_KEY` in `backend/.env`.
2. Run `/all` to start the project.
